# HomeTeacherProto

**TutoTuto 新UI プロトタイプ**

新しいパネル型インタラクション（囲む→書く→送る）のUI検証用リポジトリ。

## 🎯 目的

- 採点後のAI深掘りUIの動線確認
- `home-teacher-core` / `drawing-common` に組み込む機能・構造の明確化

## 🚀 プロトタイプ

**[Launch Proto →](https://thousandsofties.github.io/HomeTeacherProto/)**

## 📐 UI設計方針

- 1パネルがほぼ全画面を占有
- 「囲む → 書く → 送る」の統一操作
- パンくずナビゲーションで任意の深さに戻れる
- スワイプはPDF操作に使うためパネル遷移には不使用
